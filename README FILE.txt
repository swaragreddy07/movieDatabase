README FILE:

Steps to execute the code:

Step1: Install python 3.12.2
Step2: Install the zip folder
Step3: Open a code editor like VSCode and open the downloaded file
Step5: Open the terminal in VSCode and enter the command "python clean_data.py"
Step6: In the same terminal enter the command "streamlit run search.py"
You will be directed to the streamlit app in the browser.
